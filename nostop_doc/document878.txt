next prev Tkinter Toplevel Toplevel widget used create display toplevel windows which directly managed window manager. toplevel widget have parent window them. toplevel widget used when python application needs represent some extra information, pop-up, group widgets window. toplevel windows have title bars, borders, other window decorations. syntax Toplevel widget given below. Syntax Toplevel(options) List possible options given below. Options Description represents background color window. represents border size window. cursor mouse pointer changed cursor type arrow, dot, etc. when mouse window. class_ text selected text widget exported selected window manager. this make this behavior false. font font type text inserted into widget. foreground color widget. height represents height window. relief represents type window. width represents width window, Methods methods associated with Toplevel widget given following list. Method Description deiconify() This method used display window. frame() used show system dependent window identifier. group(window) used this window specified window group. iconify() used convert toplevel window into icon. protocol(name, function) used mention function which will called specific protocol. state() used current state window. Possible values normal, iconic, withdrawn, icon. transient([master]) used convert this window transient window (temporary). withdraw() used delete window doesn't destroy maxsize(width, height) used declare maximum size window. minsize(width, height) used declare minimum size window. positionfrom(who) used define position controller. resizable(width, height) used control whether window resizable not. sizefrom(who) used define size controller. title(string) used define title window.
