Syntax Frame(parent, options) list possible options given below. Option Description represents border width. background color widget. cursor mouse pointer changed cursor type different values like arrow, dot, etc. height height frame. highlightbackground color background color when under focus. highlightcolor text color when widget under focus. highlightthickness specifies thickness around border when widget under focus. relief specifies type border. default value FLAT. width represents width widget.
