Listbox(parent, options) list possible options given below. Option Description background color widget. represents size border. Default value pixel. cursor mouse pointer will look like cursor type like dot, arrow, etc. font font type Listbox items. color text. height represents count lines shown Listbox. default value highlightcolor color Listbox items when widget under focus. highlightthickness thickness highlight. relief type border. default SUNKEN. selectbackground background color that used display selected text. selectmode used determine number items that selected from list. BROWSE, SINGLE, MULTIPLE, EXTENDED. width represents width widget characters. xscrollcommand used user scroll Listbox horizontally. yscrollcommand used user scroll Listbox vertically. Methods There following methods associated with Listbox. Method Description activate(index) used select lines specified index. curselection() returns tuple containing line numbers selected element elements, counting from nothing selected, returns empty tuple. delete(first, last None) used delete lines which exist given range. get(first, last None) used list items that exist given range. index(i) used place line with specified index widget. insert(index, *elements) used insert lines with specified number elements before specified index. nearest(y) returns index nearest line coordinate Listbox widget. see(index) used adjust position listbox make lines specified index visible. size() returns number lines that present Listbox widget. xview() This used make widget horizontally scrollable. xview_moveto(fraction) used make listbox horizontally scrollable fraction width longest line present listbox. xview_scroll(number, what) used make listbox horizontally scrollable number characters specified. yview() allows Listbox vertically scrollable. yview_moveto(fraction) used make listbox vertically scrollable fraction width longest line present listbox. yview_scroll (number, what) used make listbox vertically scrollable number characters specified.
