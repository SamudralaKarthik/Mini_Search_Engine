Methods Menu widget contains following methods. Option Description add_command(options) used Menu items menu. add_radiobutton(options) This method adds radiobutton menu. add_checkbutton(options) This method used checkbuttons menu. add_cascade(options) used create hierarchical menu parent menu associating given menu parent menu. add_seperator() used seperator line menu. add(type, options) used specific menu item menu. delete(startindex, endindex) used delete menu items exist specified range. entryconfig(index, options) used configure menu item identified given index. index(item) used index specified menu item. insert_seperator(index) used insert seperator specified index. invoke(index) used invoke associated with choice given specified index. type(index) used type choice specified index.
